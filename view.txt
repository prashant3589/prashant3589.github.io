Design and implement an environment-wise Azure DevOps build pipeline for SSIS projects that can compile SSIS solutions and generate ISPAC packages for each target environment (Dev / QA / UAT / Prod).

The pipeline should:

Build SSIS projects using SSDT / MSBuild

Support environment-specific configuration through pipeline variables or variable groups

Generate ISPAC files deterministically

This pipeline will standardize SSIS builds, ensure traceability of artifacts across environments, and enable reliable downstream release/deployment pipelines.
